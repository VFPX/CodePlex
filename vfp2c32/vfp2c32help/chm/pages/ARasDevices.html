ARasDevices

VFP2C_INIT_RAS

Stores information about all available RAS-capable devices into an array.

ARasDevices(cArrayName)

Parameters

cArrayName
On return the array contains the following information from the RASDEVINFO structure.
ColumnContent
1Type of device.
2Name of device.

Return Value

The number of RAS devices.
