CancelFileChange

VFP2C_INIT_ASYNC

Stops the thread which monitors a specifed directory for file changes.

CancelFileChange(nThreadHandle)

Parameters

nThreadHandle

The thread handle returned from FindFileChange.

Return Value

.T. if the thread monitoring the directory was terminated, .F. otherwise.
