FT2DT

Converts a FILETIME struct to a datetime value.

FT2DT(nFileTimePointer [, bToLocalTime])

Parameters

nFileTimePointer

A pointer to a FILETIME structure.

bToLocalTime (optional)

default = .F.

If .T the filetime is converted to the local timezone.

Return Value

A datetime value.
