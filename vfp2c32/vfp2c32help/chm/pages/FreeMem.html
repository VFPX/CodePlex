FreeMem

VFP2C_INIT_MARSHAL

Frees a memory block allocated from the library internal heap by the AllocMem or ReAllocMem function.

FreeMem(nAddress)

Parameters

nAddress

A pointer to a memory block allocated by AllocMem.

Return Value

Always .T.
