ReAllocMem

VFP2C_INIT_MARSHAL

Changes the size of a memory block previously allocated by AllocMem.

ReAllocMem(nAddress, nNumberOfBytes)

Parameters

nAddress

A pointer to a memory block returned from AllocMem.

nNumberOfBytes

The new size of the memory block.

Return Value

A pointer (numeric) to the requested memory block.
