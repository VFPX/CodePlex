DT2FT

Converts a datetime value into a FILETIME struct.

DT2FT(tTime, nFileTimePointer [, bIsUTC])

Parameters

tTime

A datetime value.

nFileTimePointer

A pointer to a FILETIME structure.

bIsUTC (optional)

default = .F.

If .T. the passed datetime is assumed to be already in UTC time.

Return Value

Always .T.
