SyncToSNTPServer

VFP2C_INIT_WINSOCK

Syncronizes the system time and date with a SNTP server.

SyncToSNTPServer(cServer [, nPort [, nTimeout]])

Parameters

cServer

The hostname of the SNTP server.

nPort (optional)

default = 123

The port on which the SNTP service is listening.

nTimeout (optional)

default = 4000

The timeout in milliseconds.

Return Value

Always .T.
