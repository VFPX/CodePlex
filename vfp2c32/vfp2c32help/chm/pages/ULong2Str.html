ULong2Str

Converts a unsigned long (32bit numeric value) value into a binary string.

ULong2Str(nValue)

Parameters

nValue

A numeric value in the range 0 to 4,294,967,295.

Return Value

A string which is binary equal to the passed in unsigned long value.
