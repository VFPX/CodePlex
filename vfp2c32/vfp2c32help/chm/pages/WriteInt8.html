WriteInt8

VFP2C_INIT_MARSHAL

Writes a 8-bit integer at the specified address.

WriteInt8(nAddress, nValue)

Parameters

nAddress

The memory address to which the data should be written.

nValue

A numeric value in the range –128 to 127.

Return Value

Always .T.
