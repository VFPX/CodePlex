GetSystemTime

Retrieves the current system date and time either in Coordinated Universal Time (UTC)
or in local time.

GetSystemTime([bUTCTime])

Parameters

bUTCTime (optional)

default = .F.

If .T. the time is returned in UTC.

Return Value

A datetime value.
