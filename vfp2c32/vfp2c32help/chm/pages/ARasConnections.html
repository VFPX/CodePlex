ARasConnections

VFP2C_INIT_RAS

Stores information about all active RAS connections into an array.

ARasConnections(cArrayName)

Parameters

cArrayName
On return the array contains the following information from the RASCONN and RASPPPIP structures.
ColumnContent
1The name of the dialup connection.
2The device name.
3The device type.
4IP address.
5HRASCONN handle.

Return Value

The number of RAS connections.
