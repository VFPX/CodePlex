DeleteFileEx

Deletes a file.

DeleteFileEx(cFilename)

Parameters

cFileName

The fully qualified path of the file.

Return Value

Always .T.

Remarks

The function removes the readonly attribute which prevents deletion with the VFP DELETE FILE command beforehand if necessary.
