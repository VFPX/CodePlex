ASplitStr

Splits a string based on a supplied length into an array.

ASplitStr(cArrayName, cString, nLen)

Parameters

cArrayName

The name of the array into which the string should be split.

cString

The string to split.

nLen

The length of each substring.

Return Value

The number of substrings.

Example

Copy code
LOCAL lcString, lnCount, laSubString[1]
m.lcString = '12345678901234567890123456789012345'
m.lnCount = ASplitStr('laSubString', m.lcString, 10)
DISPLAY MEMORY LIKE laSubString
